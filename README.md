# 皮尔森心理原型测试 · 公益版

一个基于《内在英雄》（卡罗尔·S·皮尔森 著）核心理论的在线自测工具。采用杂志风尚的视觉风格，帮助你在 6 种心理原型（孤儿、流浪者、战士、利他主义者、天真者、魔法师）中找到当前阶段的主导原型。

🔗 **在线体验地址**：[https://fayefayeryana.github.io/archetype-test/](https://fayefayeryana.github.io/archetype-test/)

---

## ✨ 功能特点

- **双模式测评**：支持“自我评估”和“他人评估”两种视角，对比不同角度下的原型倾向。
- **即时反馈**：勾选完特质后一键生成结果，直观呈现 A（积极）与 B（消极）得分柱状图。
- **深度解读**：结果页直接展示书中关于该原型的“情节结构”、“天赋”与核心原文段落，帮助理解原型的积极面和阴影面。
- **并列处理**：当多个原型得分完全一致时，自动并列展示并分别提供解读，不回避复杂情况。
- **杂志视觉**：高对比黑白红配色、衬线字体与直角设计，呈现简洁锐利的阅读体验。

## 🧭 使用说明

1. 选择“自我评估”或“他人评估”模式。
2. 在 6 组题目中，分别从 A 栏（积极特质）和 B 栏（消极特质）勾选符合自身情况的选项。
3. 点击“查看结果”按钮，系统将自动计算并展示你的主导原型及深度解读。
4. 点击“重置”可清空所有选项，重新测试。

> 💡 该页面完全运行在浏览器端，**不会收集、存储或上传任何个人数据**，关闭页面即清空，可放心使用。
